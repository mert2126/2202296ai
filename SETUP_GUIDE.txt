CUSTOMIZATION
=============

To modify hyperparameters, edit src/config.py:

PPOHyperparameters class:
  learning_rate = 5e-4  → Adjust for faster/slower learning
  gamma = 0.8           → Safety vs. reward horizon tradeoff
  n_steps = 1024        → Batch size (increase for stability, decrease for memory)
  batch_size = 64       → Mini-batch size (CPU prefers smaller batches)

To change total training duration, modify TrainingConfig:
  total_timesteps: int = 100_000  → Increase for longer training (more time needed)

To adjust video generation, modify EvaluationConfig:
  max_episodes = 3      → Number of episodes per stage
  video_fps = 15        → Frames per second (increase for smoother video)
